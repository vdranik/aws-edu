*IP adressing*
Адрес инстанса в интренете
Бывают:
 - private
   по умолчанию у всех инстансов private IP adresses
   такой адрес дает возможность инстансам взаимодействовать друг с другом, если они нах. внутри одногот VPC
 - public
   инстансы запускаются с public IP adresses по умолчанию в зависимости от настроек VPC и подсети (VPC, который был создан амазоном по умолчанию, имеет такие настройки)
   тапой тип адреса нужен для выхода инстансом в интернет


При создании ЕС2, на вскладке Configure Instance, настройка Auto-assign Public IP будет включена, т.к. будет смотреть на стандартный VPC Network.

Весь путь (если есть проблема выхода в интернет, нужно проверять все части)
EC2 --- IP Adress --- SG --- NACL --- Route table --- Internet Gateway --- Intenet
